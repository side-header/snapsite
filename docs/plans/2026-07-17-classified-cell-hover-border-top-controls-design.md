# 분류된 사진 셀 호버 테두리와 상단 버튼 설계

## 목표

분류된 영역의 사진 셀에 마우스를 올리면 셀 테두리를 파란색으로 강조하고, 셀 조작 버튼을 셀 내부 상단에 표시한다.

## 적용 범위

- 사진이 들어 있는 셀과 빈 셀에 동일하게 적용한다.
- 대상 영역의 `− / ← / →` 버튼과 제외 영역의 `−` 버튼에 동일한 위치 규칙을 적용한다.

## 스타일 및 배치

- 평상시 셀 테두리는 기존 회색 `#d8dde2`를 유지한다.
- 호버 시 실제 셀 테두리를 파란색 `#2f80ed` 1px로 표시한다.
- 버튼 영역은 셀 내부 하단 중앙에서 상단 중앙으로 이동한다.
- 버튼 영역의 상단 여백은 5px로 한다.
- 기존 버튼 스타일, 셀 크기와 둥근 모서리는 유지한다.

## 구현 방식

채워진 셀과 빈 셀이 공통으로 사용하는 호버 래퍼가 실제 셀 `Border`를 직접 받아 테두리 색을 변경하도록 한다. 기존 호버 상태에서 버튼 표시와 테두리 변경을 함께 처리한다.

## 검증

- 채워진 셀과 빈 셀 호버 시 파란 테두리가 표시되는지 확인한다.
- 호버를 해제하면 회색 테두리로 돌아오는지 확인한다.
- 버튼 영역이 셀 내부 상단 중앙에 표시되는지 확인한다.
- 버튼 클릭, 사진 드래그, 미리보기와 셀 삽입·삭제 동작이 유지되는지 확인한다.
- 프로젝트 빌드와 앱 재실행을 확인한다.
