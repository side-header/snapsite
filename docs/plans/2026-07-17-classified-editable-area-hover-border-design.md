# 분류된 편집 영역 호버 테두리 설계

## 목표

분류된 영역의 공종 이름, 사진 제목과 사진 파일 이름에 마우스를 올렸을 때 해당 영역의 테두리를 파란색으로 표시한다.

## 적용 대상

- 공종 이름의 표시 및 편집 영역
- `전/중/후` 등 사진 셀 제목의 표시 및 편집 영역
- 사진 파일 이름의 표시 및 편집 영역

## 스타일

- 평상시에는 투명한 1px 테두리를 유지한다.
- 호버 시 파란색 `#2f80ed` 1px 테두리로 변경한다.
- 모서리 반경은 4px로 한다.
- 텍스트가 실제로 표시되는 영역만 감싸고 열이나 셀 전체의 빈 공간은 포함하지 않는다.

## 구현 방식

표시 컨트롤을 감싸는 공통 호버 테두리 헬퍼를 추가한다. 분류된 영역의 세 호출 지점에서만 이 래퍼를 사용하며, 내부 표시 컨트롤과 편집 컨트롤을 그대로 유지한다.

## 검증

- 세 영역의 표시 상태와 편집 상태에서 호버 테두리가 나타나는지 확인한다.
- 마우스를 벗어나면 테두리가 투명해지는지 확인한다.
- 공종 이름, 사진 제목과 파일 이름의 클릭 편집 및 Enter 저장이 유지되는지 확인한다.
- 사진 드래그와 미리보기 동작이 유지되는지 확인한다.
- 프로젝트 빌드와 앱 재실행을 확인한다.
