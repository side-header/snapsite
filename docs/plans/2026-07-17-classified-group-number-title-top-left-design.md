# 분류된 공종 번호와 제목 좌측 상단 배치 설계

## 목표

각 공종 영역의 가장 좌측 상단에 공종 번호를 표시하고, 공종 제목을 번호 바로 오른쪽에 같은 상단 기준으로 배치한다.

## 배치

- 공종 번호 열의 폭을 36px로 줄여 번호와 제목 사이의 불필요한 간격을 제거한다.
- 번호는 왼쪽 8px, 위쪽 7px 위치에 고정한다.
- 제목 컨테이너는 번호 열 바로 오른쪽에서 위쪽 7px 기준으로 정렬한다.
- 호버 삽입 버튼은 번호 오른쪽에 플로팅하며 기본 번호·제목 위치를 이동시키지 않는다.
- 페이지당 사진 수와 삭제 버튼의 우측 배치는 유지한다.

## 구현 방식

기존 공종 헤더 그리드의 첫 번째 열 폭과 번호 호스트 폭을 함께 줄인다. 제목 컨테이너에 상단 정렬과 위쪽 여백을 지정해 번호와 제목을 같은 기준선에 둔다.

## 검증

- 일반 공종과 빈 페이지에서 번호가 각 영역의 좌측 상단에 표시되는지 확인한다.
- 제목이 번호 바로 오른쪽 상단에 표시되는지 확인한다.
- 제목 편집, 호버 삽입 버튼, 페이지당 사진 수와 삭제 버튼 동작이 유지되는지 확인한다.
- 프로젝트 빌드와 실행 중인 앱 상태를 확인한다.
