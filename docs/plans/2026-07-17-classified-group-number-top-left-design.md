# 분류된 공종 번호 좌측 상단 배치 설계

## 목표

분류된 영역의 공종 번호를 일반 공종과 빈 페이지 모두에서 행의 좌측 상단에 고정한다.

## 배치

- 공종 번호는 행 왼쪽에서 10px, 위쪽에서 8px 위치에 표시한다.
- 평상시와 호버 상태에서 공종 번호의 위치는 바뀌지 않는다.
- 페이지 삽입 버튼 오버레이는 번호 오른쪽의 상단에 표시한다.
- 기존 버튼 문구, 스타일과 삽입 동작은 유지한다.

## 구현 방식

공종 번호 텍스트의 세로 정렬을 상단으로 변경하고 여백을 지정한다. 호버 오버레이도 상단 정렬과 대응하는 여백을 사용한다.

## 검증

- 일반 공종과 빈 페이지 번호가 좌측 상단에 표시되는지 확인한다.
- 호버 전후 번호가 이동하지 않는지 확인한다.
- 호버 버튼이 번호 오른쪽 상단에 표시되는지 확인한다.
- 프로젝트 빌드가 성공하는지 확인한다.
