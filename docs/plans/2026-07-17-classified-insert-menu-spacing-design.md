# 공종 번호와 삽입 메뉴 간격 조정 설계

## 목표

공종 번호와 번호 호버 시 표시되는 세로 삽입 버튼 영역을 더 가깝게 배치한다.

## 배치

- 삽입 메뉴 시작 위치를 기존 `(왼쪽 28px, 위쪽 26px)`에서 `(왼쪽 24px, 위쪽 22px)`로 이동한다.
- 번호 숫자 자체는 가리지 않는다.
- 파란 번호 테두리의 오른쪽 아래 모서리와 메뉴 테두리가 자연스럽게 연결되도록 일부만 겹친다.
- 버튼 문구, 크기, 세로 간격과 삽입 동작은 유지한다.

## 검증

- 메뉴가 이전보다 번호에 가깝게 표시되는지 확인한다.
- 번호 숫자가 메뉴에 가려지지 않는지 확인한다.
- 번호에서 메뉴로 마우스를 이동할 때 호버 상태가 유지되는지 확인한다.
- 프로젝트 빌드와 앱 재실행을 확인한다.
