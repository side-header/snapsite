# 공종 번호 호버 테두리 설계

## 목표

분류된 영역의 공종 번호에 마우스를 올렸을 때 번호 영역을 파란 테두리로 강조한다.

## 스타일

- 공종 번호를 28×28px 영역으로 감싼다.
- 평상시에는 테두리를 투명하게 표시한다.
- 호버 시 파란색 `#2f80ed` 1px 테두리를 표시한다.
- 모서리 반경은 4px로 한다.
- 번호와 공종 제목의 현재 위치는 유지한다.

## 호버 상태

번호 호버로 세로 삽입 메뉴가 열린 뒤 메뉴 위로 마우스를 이동해도 파란 테두리를 유지한다. 번호 및 메뉴 영역을 모두 벗어나면 테두리와 메뉴를 함께 숨긴다.

## 검증

- 평상시 번호 테두리가 보이지 않는지 확인한다.
- 번호 호버 시 파란 테두리가 표시되는지 확인한다.
- 메뉴로 이동해도 테두리와 메뉴가 유지되는지 확인한다.
- 프로젝트 빌드와 앱 재시작을 확인한다.
