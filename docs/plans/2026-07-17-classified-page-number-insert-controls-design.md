# 분류된 페이지 번호 삽입 버튼 설계

## 목표

분류된 영역의 페이지 번호에 마우스를 올리면 현재 페이지 바로 위·아래에 빈 페이지 또는 공종 페이지를 삽입하는 버튼을 표시한다.

## 호버 UI

- 일반 공종 페이지와 빈 페이지의 번호에 모두 동일하게 표시한다.
- 번호는 평소 그대로 유지하고, 호버하면 번호 옆에 `↑ 빈 페이지`, `↓ 빈 페이지`, `↑ 공종 페이지`, `↓ 공종 페이지` 버튼을 가로로 표시한다.
- 행 높이와 제목 위치는 변경하지 않는 플로팅 오버레이로 표시한다.
- 사진 셀 호버 버튼과 유사한 흰색 배경, 회색 테두리, 둥근 모서리 스타일을 사용한다.

## 삽입 동작

- `↑ 0`: 현재 페이지 바로 앞에 빈 페이지를 삽입한다.
- `↓ 0`: 현재 페이지 바로 뒤에 빈 페이지를 삽입한다.
- `↑ 공종`: 현재 페이지 바로 앞에 기본 셀이 준비된 공종 페이지를 삽입한다.
- `↓ 공종`: 현재 페이지 바로 뒤에 기본 셀이 준비된 공종 페이지를 삽입한다.
- 공종 페이지를 삽입하면 새 공종을 선택 상태로 만든다.
- 빈 페이지를 삽입하면 기존 공종 선택 상태를 유지한다.

## 상태 및 저장

상태 계층에 지정한 목록 인덱스로 빈 페이지와 공종 페이지를 삽입하는 API를 추가한다. UI는 현재 행의 인덱스를 기준으로 앞 또는 뒤 인덱스를 전달한다. 삽입 후 메타데이터를 저장하고 전체 화면을 갱신하며, 표시 번호는 변경된 목록 순서에 따라 다시 계산한다.

## 검증

- 첫 페이지 위와 마지막 페이지 아래 삽입을 확인한다.
- 일반 공종과 빈 페이지를 기준으로 네 가지 삽입 동작을 확인한다.
- 공종의 기본 셀 초기화와 선택 상태를 확인한다.
- 고유 공종 ID가 연속 삽입에서도 충돌하지 않는지 확인한다.
- 프로젝트 빌드가 성공하는지 확인한다.
