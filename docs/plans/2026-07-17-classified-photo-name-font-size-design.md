# 공종 제목과 사진 파일명 글자 크기 통일 디자인

## 목표

공종 제목과 분류된·분류되지 않은 사진 파일명의 가독성을 높이고 `한 페이지당 사진 수` 문자열과 같은 크기로 통일한다.

## 설계

- 공통 글자 크기 상수를 `14`로 정의한다.
- 공종 제목 TextBox에 공통 글자 크기를 적용한다.
- 분류된 사진 파일명 TextBox에 공통 글자 크기를 적용한다.
- 분류되지 않은 사진 파일명 TextBox에 공통 글자 크기를 적용한다.
- `한 페이지당 사진 수` TextBlock에도 같은 상수를 명시적으로 적용한다.
- 텍스트 박스의 높이, 너비, 정렬과 편집 동작은 변경하지 않는다.

## 검증

- 솔루션 전체를 빌드한다.
- 기존 특성 테스트를 실행해 분류와 저장 동작에 영향이 없는지 확인한다.
- 앱을 재실행해 변경된 UI가 실행되는 상태로 둔다.
