# 분류된 영역 팝업 버튼 호버 디자인

## 목표

분류된 영역의 공종 번호 팝업과 사진 셀 팝업에서 버튼에 마우스를 올렸을 때 현재 조작 대상을 분명하게 표시한다.

## 설계

- 두 팝업이 공통으로 사용하는 `CellControlButton`의 상태 색상만 변경한다.
- 기본 상태의 배경색과 테두리 색상은 기존 값을 유지한다.
- 마우스 오버 상태는 연한 하늘색 배경 `#eaf4ff`와 파란색 테두리 `#2f80ed`를 사용한다.
- 누름 상태는 조금 더 진한 하늘색 배경 `#d8ecff`와 파란색 테두리 `#2f80ed`를 사용한다.
- 버튼의 크기, 배치, 글자, 클릭 동작은 변경하지 않는다.
- 다른 화면이나 일반 버튼의 스타일에는 영향을 주지 않는다.

## 검증

- 솔루션 전체를 빌드한다.
- 기존 특성 테스트를 실행해 분류 및 저장 동작에 영향이 없는지 확인한다.
- 앱을 재실행해 변경된 코드가 실행되는 상태로 둔다.
