# 분류된 영역 팝업 안내 메시지 호버 유지 디자인

## 목표

공종 번호와 사진 셀의 버튼 팝업에서 안내 메시지로 마우스를 이동해도 버튼 팝업이 닫히지 않게 한다.

## 설계

- 버튼의 기본 툴팁 대신 버튼 팝업과 같은 시각 트리에 속하는 안내 상자를 사용한다.
- 버튼 위에 마우스를 올리면 해당 버튼의 안내 문구를 가까운 위치에 표시한다.
- 버튼, 안내 상자, 버튼 팝업을 하나의 포인터 영역으로 취급한다.
- 안내 상자 위에 마우스가 있는 동안 버튼 팝업과 안내 상자를 유지한다.
- 안내 상자와 버튼 팝업을 모두 벗어나면 안내 상자와 버튼 팝업을 숨긴다.
- 공종 번호 팝업과 사진 셀 팝업에만 적용하며 버튼의 클릭 동작과 기존 색상은 유지한다.

## 검증

- 솔루션 전체를 빌드한다.
- 기존 특성 테스트를 실행해 분류와 저장 동작에 영향이 없는지 확인한다.
- 앱을 재실행해 변경된 UI가 실행되는 상태로 둔다.
