# 분류된 제목 세로 중앙 정렬 설계

## 목표

분류된 영역의 공종 제목과 사진 셀 제목 글자가 각각의 호버 테두리 안에서 세로 중앙에 표시되도록 한다.

## 적용 방식

- 공종 제목 표시 `TextBlock`과 편집 `TextBox`에 `LineHeight=18`을 적용한다.
- 사진 셀 제목 `TextBox`에 `LineHeight=20`을 적용한다.
- 기존 높이 28px와 가로 정렬을 유지한다.
- 고정 줄 높이를 사용해 글꼴이나 배율 변화에도 위·아래 기준이 안정적으로 유지되게 한다.

## 유지 범위

- 호버 파란 테두리와 편집 동작을 유지한다.
- 공종 제목 영역의 상단 배치와 사진 셀 전체 배치는 변경하지 않는다.
- 사진 파일 이름과 다른 텍스트 영역은 변경하지 않는다.

## 검증

- 공종 제목 표시·편집 상태의 세로 기준선이 같은지 확인한다.
- 셀 제목 글자가 호버 테두리 안에서 세로 중앙에 보이는지 확인한다.
- 클릭 편집과 Enter 저장 동작이 유지되는지 확인한다.
- 프로젝트 빌드와 앱 재실행을 확인한다.
