# 상위 폴더 상태 개수 누적 설계

## 목표

분류되지 않은 영역의 폴더 줄에 표시하는 선택·분류 사진 개수를 해당 폴더 바로 아래 사진뿐 아니라 모든 하위 폴더 사진까지 합산한다.

## 집계 규칙

- 각 사진은 자신이 직접 속한 폴더와 경로상의 모든 상위 폴더에 한 번씩 집계한다.
- 기준 폴더 루트도 모든 하위 사진을 합산한다.
- 선택 사진과 분류 사진에 동일한 집계 규칙을 적용한다.
- 분류 사진은 현재 기준 폴더에서 스캔된 사진 중 실제 공종에 배치된 사진만 집계한다.

## 표시 규칙

- 기존 `N개 선택됨 · M개 분류됨` 형식을 유지한다.
- 개수가 0인 항목은 숨기고, 두 개수가 모두 0이면 상태 문구를 숨긴다.
- 폴더의 펼침·접힘 상태와 관계없이 누적 개수를 표시한다.

## 구현 방식

사진 경로를 한 번 순회하면서 직접 부모 폴더부터 루트까지 각 폴더 키의 개수를 증가시킨다. 선택 사진과 분류 사진은 같은 누적 집계 함수를 사용한다.

## 검증

- 폴더 바로 아래 사진이 해당 폴더와 루트에 집계되는지 확인한다.
- 한 단계 및 여러 단계 하위 폴더의 사진이 모든 상위 폴더에 한 번씩 집계되는지 확인한다.
- 서로 다른 하위 폴더의 개수가 상위 폴더에서 합산되는지 확인한다.
- 선택·분류 표시 조합과 프로젝트 빌드를 확인한다.
