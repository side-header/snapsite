# 공종 작업 팝업 hover 경계 설계

## 목적

공종 번호와 페이지 추가 팝업 밖으로 마우스를 이동해도 팝업이 닫히지 않는 문제를 해결한다.

## 동작

- 공종 번호와 팝업의 3px 겹침은 유지해 두 영역 사이를 끊김 없이 이동할 수 있게 한다.
- 공종 번호 또는 실제 팝업 테두리 위에 포인터가 있을 때만 팝업을 표시한다.
- 두 영역 모두에서 포인터가 벗어나면 즉시 팝업을 닫는다.
- 드래그 시작 시 팝업과 번호 테두리를 숨기는 기존 동작은 유지한다.

## 구현 범위

두 컨트롤을 감싸는 큰 컨테이너의 투명 hit-test 배경을 제거한다. 컨테이너 크기는 배치에 사용하되 빈 공간은 hover 대상으로 취급하지 않는다.

## 검증

- 관련 테스트 프로젝트가 없으므로 전체 애플리케이션을 빌드한다.
- 앱을 재시작해 정상 실행 여부를 확인한다.
- 번호에서 팝업으로 이동, 팝업 바깥으로 이탈, 드래그 시작의 상태 전환을 코드 경로로 확인한다.
