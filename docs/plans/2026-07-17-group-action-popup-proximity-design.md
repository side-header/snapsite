# 공종 번호 작업 팝업 간격 개선 설계

## 목적

공종 번호에서 페이지 추가 버튼 팝업으로 마우스를 이동할 때 팝업이 너무 빨리 닫히는 문제를 해결한다.

## 동작

- 팝업의 우측 아래 배치는 유지한다.
- 팝업의 좌측 상단을 공종 번호 테두리와 약 3px 겹쳐 시각적·입력상 빈틈을 없앤다.
- 팝업이 열린 동안 공종 번호와 팝업을 감싸는 영역 전체를 투명 hover 영역으로 사용한다.
- 포인터가 이 연결 영역을 완전히 벗어났을 때만 팝업을 닫는다.
- 공종 번호 드래그가 시작되면 팝업을 즉시 숨기는 기존 동작은 유지한다.

## 구현 범위

공종 번호와 페이지 추가 팝업을 포함하는 로컬 컨테이너의 위치와 hit-test 배경만 조정한다. 다른 공종 행이나 전역 포인터 상태는 변경하지 않는다.

## 검증

- 관련 테스트 프로젝트가 없으므로 전체 애플리케이션을 빌드한다.
- 앱을 재시작해 정상 실행 여부를 확인한다.
- hover 표시, 팝업으로 이동, 영역 이탈, 공종 드래그 시작의 상태 전환을 코드 경로로 확인한다.
