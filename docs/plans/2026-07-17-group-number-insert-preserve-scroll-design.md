# 공종 번호 페이지 추가 시 스크롤 유지 설계

## 목적

분류된 영역에서 공종 번호 팝업의 위·아래 빈 페이지 또는 공종 페이지 추가 버튼을 눌렀을 때 현재 스크롤 위치가 자동으로 이동하지 않게 한다.

## 동작

- 공종 번호 팝업의 네 페이지 추가 버튼은 현재 분류 영역 스크롤 위치를 유지한다.
- 새 페이지의 파란 테두리와 하늘색 배경 강조 효과는 유지한다.
- 새 페이지가 현재 화면 밖에 있으면 강조 효과는 화면에 보이지 않을 수 있다.
- 상단의 `↓ 빈 페이지`, `↓ 공종 페이지` 버튼과 선택 사진으로 공종을 추가하는 동작은 기존 자동 스크롤을 유지한다.
- 부분 갱신, 선택 공종, 자동 저장 동작은 변경하지 않는다.

## 구현

신규 공종 공개 처리에서 스크롤 이동과 강조 처리를 분리한다. 공종 번호 팝업 경로는 렌더링 이후 강조만 예약하고, 다른 생성 경로는 기존 스크롤과 강조를 함께 수행한다.

## 검증

- 전체 솔루션을 빌드한다.
- 기존 Domain/저장/내보내기 특성 검증을 실행한다.
- 앱을 재시작해 정상 실행 여부를 확인한다.
