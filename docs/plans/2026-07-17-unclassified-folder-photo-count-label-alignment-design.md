# 분류되지 않은 폴더 사진 개수 라벨 우측 정렬 설계

## 목표

사진 목록 위에 표시되는 `{폴더 이름} 폴더의 사진 총 N개` 라벨을 좌측 상단에서 우측 상단으로 이동한다.

## 배치

- 라벨 자체를 사진 목록 영역의 오른쪽에 정렬한다.
- 오른쪽 여백은 5px, 위쪽 여백은 2px로 유지한다.
- 폴더 깊이에 따른 왼쪽 들여쓰기는 라벨에 적용하지 않는다.
- 기존 문구, 글자 크기, 폴더 이름의 검은색과 안내 문구의 회색 스타일은 유지한다.

## 구현 방식

라벨의 가로 정렬을 `Right`로 지정하고 여백을 오른쪽 기준으로 조정한다. 텍스트 정렬도 오른쪽으로 지정해 여러 너비에서도 같은 방향으로 표시한다.

## 검증

- 라벨이 사진 목록 위쪽의 오른쪽 끝에 표시되는지 확인한다.
- 기존 색상과 일반 굵기 스타일이 유지되는지 확인한다.
- 프로젝트 빌드가 성공하는지 확인한다.
