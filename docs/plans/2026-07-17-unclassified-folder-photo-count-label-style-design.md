# 분류되지 않은 폴더 사진 개수 라벨 스타일 설계

## 목표

분류되지 않은 영역의 사진 목록 위에 표시되는 `{폴더 이름} 폴더의 사진 총 N개` 문구를 역할에 따라 두 가지 색상으로 구분한다.

## 스타일

- `{폴더 이름}`은 굵게 표시하지 않고 검은색 `#263238`로 표시한다.
- ` 폴더의 사진 총 N개`는 일반 굵기의 회색 `#7d8790`으로 표시한다.
- 기존 글자 크기, 위치, 좌측 정렬, 여백과 문구는 유지한다.

## 구현 방식

기존 단일 `TextBlock` 안의 두 `Run`에 각각 전경색과 일반 굵기를 지정한다. 별도 컨트롤로 나누지 않아 현재 레이아웃과 줄 흐름을 유지한다.

## 검증

- 폴더 이름에 굵은 글씨가 적용되지 않는지 확인한다.
- 폴더 이름과 나머지 안내 문구가 각각 검은색과 회색으로 표시되는지 확인한다.
- 프로젝트 빌드가 성공하는지 확인한다.
