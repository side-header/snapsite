# 분류되지 않은 사진 파일명 영구 TextBox 설계

## 목표

분류되지 않은 영역의 모든 사진 카드 파일 이름을 분류된 사진 셀과 동일한 영구 `TextBox` 구조로 표시하고, 호버 시 파란 테두리를 표시한다.

## 적용 범위

- 미분류 사진 카드와 이미 분류되어 흐리게 표시되는 사진 카드 모두에 적용한다.
- 분류된 영역과 같은 영구 파일명 `TextBox` 생성 로직을 재사용한다.

## 표시 및 편집

- 높이 24px, 한 줄, 가운데 정렬을 사용한다.
- 평상시에는 입력선이 보이지 않고 클릭하면 같은 위치에 입력선이 나타난다.
- Enter로 파일 이름을 저장하고, 포커스를 잃으면 저장하지 않은 값을 기존 파일 이름으로 복구한다.
- 파일 이름 영역 호버 시 파란색 `#2f80ed` 1px 테두리를 표시한다.

## 사진 선택 연동

- 일반 클릭은 기존처럼 사진을 선택한 뒤 파일명 편집 포커스를 준다.
- Shift/Ctrl/Command 클릭은 사진 선택만 수행하고 파일명 편집 포커스를 주지 않는다.
- 파일명 입력창의 포인터 이벤트가 카드 선택 이벤트로 중복 전달되지 않게 처리한다.
- 이미 분류된 카드의 파일명은 편집할 수 있지만 사진 선택 상태에는 추가하지 않는다.

## 흐린 카드 표시

이미 분류된 카드의 이미지와 파일명 글자는 기존처럼 흐리게 표시한다. 호버 테두리는 흐림 효과 밖에 두어 파란색이 선명하게 보이게 한다.

## 검증

- 미분류 및 이미 분류된 카드의 파일명이 항상 `TextBox`인지 확인한다.
- 일반 클릭, 수정키 클릭과 파일명 편집이 서로 중복 실행되지 않는지 확인한다.
- Enter 저장과 포커스 해제 복구를 확인한다.
- 파란 호버 테두리와 흐린 카드 표시를 확인한다.
- 프로젝트 빌드와 앱 재실행을 확인한다.
