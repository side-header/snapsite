# 분류되지 않은 사진의 분류 영역 드롭 미리보기 디자인

## 목표

분류되지 않은 영역의 사진을 분류된 영역으로 드래그할 때 실제로 삽입되거나 채워질 위치를 명확하게 표시한다.

## 드래그 표시

- 사진이 있는 셀 위에서는 포인터가 셀의 좌측인지 우측인지에 따라 해당 경계에 파란 세로선을 표시한다.
- `image == ""`인 빈 셀 위에서는 삽입선을 숨기고 셀 테두리를 파란색, 배경을 연한 하늘색으로 표시한다.
- 셀이 하나도 없는 대상 또는 나머지 영역에서는 영역 가운데에 파란 세로선을 표시한다.
- 드래그가 대상 영역을 벗어나거나 드롭이 끝나면 삽입선과 빈 셀 강조를 모두 제거한다.
- 기존 분류 영역 내부 셀 이동의 표시와 동작은 유지한다.

## 드롭 동작

- 사진이 있는 셀 경계에 드롭하면 표시된 경계 인덱스에 사진을 삽입한다.
- 여러 장은 선택 순서대로 연속 삽입한다.
- 빈 셀에 드롭하면 해당 셀부터 사용 가능한 빈 셀을 순서대로 채우고 남는 사진은 기존 규칙에 따라 추가한다.
- 셀이 없는 영역에 드롭하면 사진들을 선택 순서대로 해당 영역에 추가한다.

## 검증

- 경계 인덱스 삽입과 순서 보존을 특성 테스트로 검증한다.
- 빈 셀 우선 채우기 동작을 특성 테스트로 검증한다.
- 솔루션 전체 빌드와 기존 특성 테스트를 실행한다.
- 앱을 재실행해 변경된 UI가 실행되는 상태로 둔다.
