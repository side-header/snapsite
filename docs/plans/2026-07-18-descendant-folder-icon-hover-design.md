# 하위 폴더 아이콘 동시 호버 디자인

## 목표

분류되지 않은 영역에서 폴더 아이콘에 마우스를 올리면 해당 폴더와 현재 화면에 보이는 모든 하위 폴더 아이콘을 함께 파란색으로 표시한다.

## 설계

- 분류되지 않은 트리에 렌더링된 폴더 아이콘을 폴더 경로별로 등록한다.
- 폴더 아이콘에 마우스를 올리면 해당 폴더 경로와 같거나 그 아래 경로인 등록 아이콘을 파란색으로 변경한다.
- 접힌 폴더는 자동으로 펼치지 않으며 화면에 렌더링된 아이콘만 갱신한다.
- 마우스가 폴더 아이콘을 벗어나면 파란색 표시를 제거하고 밝은 황토색으로 복원한다.
- 부모에서 보이는 하위 폴더 아이콘으로 이동하면 새로 호버한 하위 폴더를 기준으로 범위를 갱신한다.
- 폴더 아이콘 클릭의 규칙1 선택 범위와 동일한 경로 포함 규칙을 사용한다.
- 탐색기 영역 폴더 아이콘에는 적용하지 않는다.
- 폴더 펼치기 상태와 스크롤 위치는 변경하지 않는다.

## 검증

- 기존 폴더 경로 범위 특성 테스트를 실행한다.
- 솔루션 전체를 빌드한다.
- 앱을 재실행해 변경된 UI가 실행되는 상태로 둔다.
