# 빈 사진 셀 우측 삽입 영역 디자인

## 목표

분류되지 않은 사진을 빈 사진 셀의 우측으로 드래그할 때 빈 셀을 채우는 대신 해당 셀 다음 위치에 삽입할 수 있게 한다.

## 설계

- 빈 사진 셀 슬롯의 중앙 80%는 기존 빈 셀 채우기 영역으로 유지한다.
- 중앙 80%에서는 빈 셀 테두리를 파란색, 배경을 연한 하늘색으로 표시한다.
- 빈 사진 셀 슬롯의 오른쪽 20%는 우측 삽입 영역으로 사용한다.
- 오른쪽 20%에서는 빈 셀 강조를 끄고 셀 우측 경계에 파란 세로선을 표시한다.
- 우측 삽입 영역에 드롭하면 빈 셀은 그대로 두고 그 다음 인덱스에 사진들을 선택 순서대로 삽입한다.
- 드래그 중 경계를 오가면 빈 셀 강조와 삽입선을 즉시 전환한다.
- 사진이 있는 셀과 분류 영역 내부 셀 이동 동작은 변경하지 않는다.

## 검증

- 기존 경계 인덱스 삽입 및 빈 셀 채우기 특성 테스트를 실행한다.
- 솔루션 전체를 빌드한다.
- 앱을 재실행해 변경된 UI가 실행되는 상태로 둔다.
