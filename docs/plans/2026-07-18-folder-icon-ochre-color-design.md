# 폴더 아이콘 황토색 변경 디자인

## 목표

탐색기 영역과 분류되지 않은 영역의 파란색 폴더 아이콘을 밝은 황토색으로 변경한다.

## 설계

- 두 영역이 공통으로 사용하는 `FolderGlyph()`의 색상만 변경한다.
- 폴더 몸체는 `#d8b56a`를 사용한다.
- 폴더 상단 탭은 `#e7c980`을 사용한다.
- 폴더 테두리는 `#b8954f`를 사용한다.
- 아이콘 모양, 크기, 여백과 폴더 펼침 동작은 변경하지 않는다.
- 파일 아이콘과 다른 UI 색상에는 영향을 주지 않는다.

## 검증

- 솔루션 전체를 빌드한다.
- 기존 특성 테스트를 실행해 동작에 영향이 없는지 확인한다.
- 앱을 재실행해 변경된 UI가 실행되는 상태로 둔다.
