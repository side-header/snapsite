# 폴더 규칙1 선택 토글 디자인

## 목표

분류되지 않은 영역의 폴더 아이콘을 클릭해 규칙1 사진을 누적 선택하고, 파란색 폴더를 다시 클릭해 해당 폴더 범위의 선택만 해제한다.

## 선택 토글

- 황토색 폴더를 클릭하면 해당 폴더와 모든 하위 폴더의 미분류 규칙1 사진을 기존 규칙1 선택에 추가한다.
- 파란색 폴더를 클릭하면 해당 폴더와 모든 하위 폴더에서 규칙1로 선택된 사진을 해제한다.
- 다른 폴더의 기존 규칙1 선택은 유지해 여러 폴더를 누적 선택할 수 있게 한다.
- 일반 사진 선택 상태에서 처음 폴더 아이콘을 클릭하면 일반 선택을 정리하고 규칙1 선택 모드로 전환한다.
- 마지막 규칙1 사진까지 해제되면 규칙1 선택 모드를 종료하고 하단 안내 상자를 숨긴다.
- 이미 분류된 사진은 계속 후보에서 제외한다.

## 폴더 색상

- 폴더 자체 또는 하위 폴더에 선택된 규칙1 사진이 하나라도 있으면 해당 폴더 아이콘은 파란색을 유지한다.
- 특정 하위 폴더를 다시 클릭하면 그 폴더와 하위 범위만 황토색으로 돌아온다.
- 다른 하위 폴더에 선택이 남아 있으면 부모 폴더는 파란색을 유지한다.
- 부모 아래의 모든 규칙1 선택이 해제되면 부모 폴더도 황토색으로 돌아온다.
- 호버 중인 폴더와 현재 보이는 하위 폴더의 임시 파란색은 지속 선택 색상과 함께 표시한다.

## 검증

- 부모 폴더 선택 시 하위 사진까지 추가되는지 검증한다.
- 하위 폴더 토글 해제 시 다른 하위 선택은 유지되는지 검증한다.
- 마지막 하위 선택 해제 시 부모 범위 선택이 비는지 검증한다.
- 형제 폴더 선택을 누적할 수 있는지 검증한다.
- 솔루션 전체 빌드와 기존 특성 테스트를 실행한다.
- 앱을 재실행해 변경된 UI가 실행되는 상태로 둔다.
