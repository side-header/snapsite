# 선택 안내 상자 요약행 레이아웃 디자인

## 목표

분류되지 않은 영역 하단 선택 안내 상자의 요약 문장과 공종 추가 버튼을 한 줄에 배치하고 펼치기 아이콘을 안내 문장 왼쪽으로 이동한다.

## 설계

- 안내 상자를 두 행 Grid로 구성한다.
- 첫째 행은 `펼치기/접기 아이콘`, `안내 문장`, `선택된 사진으로 공종 추가하기` 버튼 순서로 배치한다.
- 펼치기/접기 아이콘은 안내 문장 왼쪽에 두고 아이콘과 문장 사이에 여백을 둔다.
- 공종 추가 버튼은 첫째 행에만 배치해 상세 설명을 펼쳐도 요약 문장과 같은 줄을 유지한다.
- 펼친 불릿 상세 설명은 둘째 행에서 상자 전체 너비를 사용한다.
- 공종 추가 버튼의 글자 굵기는 `Normal`로 변경한다.
- 규칙1 선택과 일반 다중 선택에 같은 레이아웃을 적용한다.
- 버튼의 크기, 색상, 호버·누름 색상과 클릭 동작은 유지한다.

## 검증

- 솔루션 전체를 빌드한다.
- 기존 특성 테스트를 실행해 선택 및 공종 추가 동작에 영향이 없는지 확인한다.
- 앱을 재실행해 변경된 UI가 실행되는 상태로 둔다.
