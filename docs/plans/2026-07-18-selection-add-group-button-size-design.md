# 선택 사진 공종 추가 버튼 크기 축소 디자인

## 목표

분류되지 않은 영역 하단의 `선택된 사진으로 공종 추가하기` 버튼을 분류된 영역의 `↓ 빈 페이지` 버튼과 같은 세로 규격으로 줄인다.

## 설계

- 규칙1 선택과 일반 다중 사진 선택에서 사용하는 공통 버튼에 동일하게 적용한다.
- 버튼 높이는 `26`으로 변경한다.
- 글자 크기는 `14`로 변경한다.
- 좌우 패딩은 `12`, 상하 패딩은 `0`으로 변경한다.
- 모서리 반경은 `4`로 변경한다.
- 기존 최소 너비 `260`을 제거하고 문구와 패딩을 기준으로 가로폭을 자동 계산한다.
- 파란 배경, 호버·누름 색상과 클릭 동작은 유지한다.
- 하단 안내 문구, 펼치기·접기 동작에는 영향을 주지 않는다.

## 검증

- 솔루션 전체를 빌드한다.
- 기존 특성 테스트를 실행해 동작에 영향이 없는지 확인한다.
- 앱을 재실행해 변경된 UI가 실행되는 상태로 둔다.
