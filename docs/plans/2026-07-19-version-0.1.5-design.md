# Version 0.1.5 Design

## Goal

Update the SiteSnap application version from `0.1.4` to `0.1.5` consistently across runtime, build, and documentation sources.

## Changes

- Set `AppInfo.Version` to `0.1.5` so runtime metadata and the application UI report the new version.
- Set the application project `Version` and `InformationalVersion` to `0.1.5` so build artifacts carry the same version.
- Update `docs/manifest.md` and `docs/requirements.md` so documented version values match the application.
- Do not change assembly compatibility, dependencies, application behavior, or persisted data formats.

## Test Impact

This is a synchronized metadata change. Keep existing tests unchanged, build the application, run the characterization suite, and search the active source and documentation paths for stale `0.1.4` values.
